# snow
javascript下雪插件

安装

`npm i any-snow -S`

使用
```
// browser
<script src="./snow.js"></script>
new $Snow(options);

// commonjs
const $Snow = require('any-snow');
new $Snow(options);

// es module
import $Snow from '$Snow';
new $Snow(options);
```
