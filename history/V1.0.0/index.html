SNOW V1.0.0

一个下雪的插件 自定义选项算多了吧
如果不打算使用自定义图片或者不打算使用内置的默认雪花图片 可以选择snow-nodefimg.js
snow-nodefimg.js和snow.js的区别在于 后者包含了一个默认雪花图片的base64字符串 而前者没有 所以文件大小上存在差距
gayhub

引用:
<scrip src="snow.js"></scrip>

使用:
var snow = new Snow();
var snow = new Snow({img: true, stay: true, speed: 30});

预设:

配置项:

  • color
    颜色 当img选项为true时不生效
    默认值: 白色(#ffffff)
    有效值: 浏览器能识别的颜色就行
  • opacity
    不透明度
    默认值: 1
    有效值: 0-1之间
  • speed
    下坠速度
    默认值: 随机0 - 10毫秒 + speed值
  • randombase
    生成速度的基值 生成速度为 0 - 300 毫秒 + 该配置的值
    默认值: 1500(毫秒)
    有效值: 自己喜欢就好
  • num
    每次随机生成的雪花数量
    默认值: 8
    有效值: 反正不搞出问题随你喜欢
  • limit
    每个Snow对象最多有多少个雪花
    默认值: 150
  • offset
    是否跟随鼠标进行偏移
    默认值: true
    有效值: 布尔值
  • img
    是否以图片形式显示雪花
    默认值: false
    有效值: 布尔值
  • imgurl
    自定义图片
    默认值: 自带的一个雪花图片
    有效值: url、blob、base64
  • stay
    是否会在底部停留
    默认值: false
    有效值: 布尔值
  • stayTime
    在底部停留的时间
    默认值: 1500 (毫秒)
  • maxSize
    点状的最大尺寸
    默认值: 4px + 0-4px的随机值
    有效值: 数值类型
  • maxSize
    图片的最大尺寸
    默认值: 30px + 0-4px的随机值
    有效值: 数值类型

方法:

  • stop
    停止下雪并且清空雪花 (对象不会被清空 需要手动赋值null)
